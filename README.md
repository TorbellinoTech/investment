# Code related to the investment section of the website

## Simple pricing model for an investment portfolio (stocks)

The file simple_pricing_info_portfolio.py contains an example computation
of a simple pricing model for the information of a portfolio. The calculation
is based on entropy, information gain, some general, public constants
(number of companies of NASDAQ, average EU wealth, ...) and other sample
data. Performance ratio w.r.t. SPY and a commission percentage are
also considered.

Usage:

python3 simple_pricing_info_portfolio.py
